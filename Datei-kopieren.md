# Datei (Dokument) kopieren in Onshape

## Methode 1: Vom Dashboard aus

1. Gehe zum **Dashboard** (Startseite von Onshape)
2. **Rechtsklick** auf das Dokument, das du kopieren möchtest
3. Wähle **"Kopie erstellen"** bzw. **"Make a copy"**
4. Neuen Namen vergeben und optional einen anderen Ordner wählen
5. Mit **"OK"** bestätigen

## Methode 2: Aus einem geöffneten Dokument

1. Öffne das gewünschte Dokument
2. Klicke oben links auf den **Dokumentnamen** (oder das Hamburger-Menü)
3. Wähle **"Kopie erstellen"** / **"Make a copy"**
4. Namen und Zielordner festlegen
5. Mit **"OK"** bestätigen

## Einzelne Tabs kopieren (Part Studio, Assembly, Drawing)

1. **Rechtsklick** auf den Tab (unten in der Tab-Leiste)
2. Wähle **"Kopieren nach..."** / **"Copy to..."**
3. Wähle das Zieldokument (dasselbe oder ein anderes)
4. Bestätigen

## Hinweise

- Die Kopie ist **unabhängig** vom Original — Änderungen betreffen nur die jeweilige Version
- Die **Versionshistorie** wird nicht mitkopiert
- **Verknüpfungen** zu anderen Dokumenten bleiben erhalten, zeigen aber weiterhin auf das Original
- Um eine bestimmte Version zu kopieren: zuerst die Version öffnen, dann kopieren
